# Targets: Research & Intelligence

Target research organized by track and type.

## Directory Structure

```
targets/
  grants/              Research on grant opportunities
  residencies/         Research on residency programs
  jobs/                Research on job positions
  profiles/            JSON profile files migrated from application tracker (44 files)
  README.md            This file
```

## profiles/

The 44 JSON files in `profiles/` are target profiles migrated from the original application tracker. Each contains organization details, eligibility criteria, and application metadata. These are the raw intelligence that informs the pipeline YAML entries in `pipeline/active/`.

## grants/ and residencies/

Markdown research files with exhaustive opportunity lists and track-level overviews.

## jobs/

Role-specific research for AI engineering and related positions.
