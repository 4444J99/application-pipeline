## Residency Project Proposal — ZKM Hertzlab

**"Governance Made Visible"** — A media installation rendering the ORGANVM system's governance processes as a real-time audiovisual experience.

The eight-organ system produces a continuous stream of governance events: tests passing or failing across 4,000+ automated checks, repositories promoted through a five-state machine (LOCAL to GRADUATED), dependency edges validated against 50 enforced constraints, constitutional rules checked against 27 specs grounded in 130 academic sources. These processes are currently invisible — they execute in CI pipelines, registries, and terminal output. This project makes them visible and audible.

The installation translates governance data into projected visual patterns and generative sound. Each governance action triggers an event: a CI workflow completing produces a tonal shift; a promotion decision cascades through the dependency graph as a visual ripple; a constitutional validation check resolves into harmonic convergence or dissonance. The 32-day zero-incident soak test becomes a sustained drone; the omega maturity scorecard (currently 7/17 criteria met) drives the installation's overall tonal character, shifting as the system matures during the residency itself.

The system is not a simulation. It renders live data from the actual production infrastructure — the same 113 repositories, 104 CI workflows, and governance rules that operate daily. Visitors experience a creative system governing itself in real time.

**Deliverables:** (1) A site-specific installation designed for ZKM's Hertzlab production facilities — projection mapping, spatial audio, real-time data feeds. (2) Four to six essays documenting the residency process, extending the existing series of 48 published essays on creative methodology. (3) An open-source visualization toolkit allowing other practitioners to render their own governance systems as audiovisual experiences.

**Why ZKM:** The Zentrum fur Kunst und Medien stands in a tradition of computational art — Peter Weibel's engagement with cybernetics, Rudolf Frieling's work at the intersection of media and institutional critique. "Governance Made Visible" belongs in this lineage: it treats institutional process itself as artistic medium, using ZKM's production facilities to make that argument at a scale impossible in a standard studio. The Hertzlab's media art production infrastructure — projection systems, spatial audio, fabrication — is specifically what this project requires.

**Timeline:** Three-month residency, October through December 2026. Studios and rehearsal rooms for iterative development; public presentation in the final weeks.
