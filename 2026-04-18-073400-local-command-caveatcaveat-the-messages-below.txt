  git layer.

  counter-first). If the push fails and the session

  protocol prevents this by separating the claim (counter                       
  commit) from the use (IRF edit).

  per-ref, so one will fail. The protocol handles it.

  greps for DONE-NNN uniqueness before allowing any commit
